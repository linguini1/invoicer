# Invoicer
### Matteo Golin

## Usage
This software creates invoices from a set of input parameters. Invoices are created in an HTML template and then 
converted to PDF format.

## Installation
Python 3.10.0 or later is required

- Beautiful Soup (bs4)
- pdfkit
- pdfkit's dependency wkhtmltopdf
- Pandas
